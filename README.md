## 分支管理

#### master

```
主分支，上线后需合并到master
```

```
开发分支，新需求分支从此分支拉取新分支
```
### Pre

```
每次开发完成，需合并到pre进行pre测试，测试完可上线（当前阶段等同于master分支）
```
